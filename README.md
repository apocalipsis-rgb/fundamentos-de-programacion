# Fundamentos-de-programacion
Para registrar y controlar las horas laboradas cada semana se desarrolla un sistema en Python de un equipo de trabajo en el cual se pueden calcular automáticamente el total de horas y clasificar la jornada de cada empleado según las horas registradas.

# Sistema Control de Horas Laborales

Proyecto desarrollado en Python para registrar las horas trabajadas por empleados durante la semana laboral.

## Funcionalidades

- Registro de empleados
- Registro de horas trabajadas por día
- Cálculo de horas semanales
- Clasificación de jornada laboral
- Identificación de sobretiempo
